Whatever You Want Vending Machine

Core Concept

Have you ever struggled to find your desired drink at a vending machine? This innovative project features an interactive vending machine capable of producing any beverage flavor you desire! Simply input your chosen flavor into the machine's dialogue box, insert a coin, wait a moment, and you will receive your custom-flavored drink. This vending machine is set to launch soon, so stay tuned!

Technical Details

This interactive vending machine is built within the Unity environment. It integrates Hugging Face AI models to create a unique and engaging user experience. Users interact with the vending machine by typing their desired flavor into a text input box. The connected AI then processes this input to instantly generate a custom canned beverage graphic representing the requested flavor. This creates a truly expressive and personalized vending experience where any flavor imaginable can be materialized.
